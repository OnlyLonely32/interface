How to play?

A random word is selected by the program and draws a certain number of underlines, how many letters are in the word. The player begins to indicate the letters to guess the word. If the letter is in the word, then it will be displayed in an empty field (if there are several such letters, then all fit), and if not, then the program draws one element of the hanged man (head, body, 2 arms, 2 legs - only 6 attempt elements).
